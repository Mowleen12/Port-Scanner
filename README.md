# 🔍 Port Scanner

A fast and minimal **Port Scanner** built with **Vite + React + TypeScript**.  
This tool allows users to scan a range of ports for a given IP address using a clean and interactive UI.

> Developed with the help of **Bolt AI** and crafted by **Mowleen Armstrong**.

---

## 🚀 Features

- Scan a range of ports on any valid IPv4 address
- Visual feedback on open/closed ports
- Responsive and intuitive UI
- Input validation for IP and port range
- Built using modern tech stack (React + TypeScript + Tailwind CSS)

---

## 🧰 Tech Stack

- **React** – UI library
- **Vite** – Lightning-fast build tool
- **TypeScript** – Type-safe development
- **Tailwind CSS** – Utility-first styling
- **Lucide React** – Icons
- **Bolt AI** – Assisted development
